# Creating your first AI Analyst

## Quick steps:

1. Connect your data in Actian AI Analyst
2. Create a first model in the semantic layer
3. Create an AI Analyst
4. Set up the Agent
   1. Link relevant models to the Agent
   2. Give it clear instructions
   3. ...
5. Test & refine your agent
6. Deploy your agent to Slack or Teams, or use it via Actian AI Analyst Explorer
