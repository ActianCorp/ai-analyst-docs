# Manage metadata in the UI

Actian AI Analyst lets you view and edit all table and column metadata directly from the web interface.

### How to Edit Metadata in the UI

1. **Open Your Data Source**
    * Click **Connections** at the bottom of the sidebar.
    * Click on the relevant database and table.
2. **View Table Metadata**
    * Select the **Metadata** tab.
    * You’ll see two main sections: **Table** and **Columns**.
3. **Edit Table-Level Metadata**
    * **Description for AI Agents:**\
     Write a clear description of the table’s business purpose, key features, and any special considerations.
    * **Common Pitfalls for AI Agents:**\
     Note any data quirks, exceptions, or edge cases that AI Agents should be aware of.
4. **Edit Column-Level Metadata**
    * Click on a column to edit its details:
     * **Description:** What the column represents, and any definitions or business logic.
     * **Data Type:** The data type (e.g., string, bigint, date).
     * **Semantic Type:** Choose the semantic category if available (e.g., NAME, EMAIL).
     * **Rules:** Add validation or rules, if needed.
     * **Visibility:** Use the toggle to hide deprecated or sensitive columns from AI Agents.
5. **Save Changes**
    * After editing, click **Save changes** to apply updates.
