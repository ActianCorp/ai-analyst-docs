# Connect a data source

Actian AI Analyst supports two types of connections:

- **Data source connections** — connect to a database or data warehouse so AI Analysts can query your data. These are described on this page and its sub-pages.
- **Catalog connections** — connect to an enterprise data catalog (such as the [Actian Data Intelligence Platform](../catalog/actian-data-intelligence-platform.md)) to automatically sync business terminology into your Glossary. See [Catalog Connections](../catalog/README.md) to learn more.
