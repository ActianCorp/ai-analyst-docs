# Actian AI Analyst Lossless Format

The native Actian AI Analyst format exports all details of your semantic layer for complete backup and restoration.

### What's Exported

* **Models**: Dimensions, measures, filters, relationships, grain, sample values
* **Metrics**: Definitions, anchors, time dimensions, joins, filters
* **Glossary**: Terms, definitions, synonyms, mappings
* **Agents**: Configurations, instructions, semantic layer access permissions

### File Types

* **YAML** - Human-readable, version control friendly
* **JSON** - Machine-readable, API integrations
* **Excel** - Spreadsheet format for review and editing
